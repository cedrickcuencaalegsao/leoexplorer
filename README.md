# Project Name: Leo Explorer

Background:
Leo Explorer began as a personal side project created by Cedrick C. Alegsao. The original goal was to address the lack of innovation, intelligence, and modern productivity features found in traditional file explorer applications across Windows, macOS, and Linux.

As development progressed, the project evolved into a thesis proposal at ACLC College of Mandaue in fulfillment of the requirements for the subjects "Design Project 1" and "Design Project 2" under the Bachelor of Science in Computer Science program.

The project was developed by two Computer Science students:

* Cedrick C. Alegsao
* Joshua Mark H. Playda

Thesis Information:
Title:
"Leo Explorer: An AI-Augmented, Multithreaded File Management Solution for Modern Operating Systems"

The project focused on creating a next-generation file management platform that integrates AI-assisted workflows, intelligent file organization, advanced search capabilities, multithreaded operations, and modern user experience principles.

Achievements:

* Started as a private repository.
* Originally developed using Tauri 2.0, Rust, and TypeScript.
* Successfully defended as a thesis project.
* Received the Best Design Project Award during the ACLC College of Mandaue Graduation Ceremony held on May 22.
* Demonstrated innovation in file management, user experience design, and AI-assisted productivity.

Current Development:
This repository represents the next evolution of Leo Explorer.

The project is currently being rebuilt and redesigned using:

* Rust
* Tauri 2.0
* Dioxus

Goals of the rewrite:

* Eliminate unnecessary complexity.
* Improve maintainability.
* Create a fully Rust-powered architecture.
* Support Windows, macOS, and Linux.
* Enable future AI-powered file management features.
* Establish a strong open-source foundation for community contributions.
